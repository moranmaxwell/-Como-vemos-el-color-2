Código base S3 - Segunda página “contenido” con JavaScript

Archivos incluidos:
- index.html: página principal del proyecto.
- contenido.html: segunda página informativa. Debe conservar este nombre exacto.
- style.css: estilos similares al código base original.
- script.js: funciones de redirección y mostrar/ocultar respuestas.
- logo-unein.png: logo institucional.
- imagen-tema.svg: imagen de ejemplo que el estudiante puede reemplazar.

Indicaciones para el estudiante:
1. Cambiar APELLIDO1 APELLIDO2 por sus apellidos.
2. Reemplazar “ESCRIBE AQUÍ TU TEMA SELECCIONADO” por el tema real.
3. Completar las tres respuestas de contenido.html.
4. Reemplazar las fuentes de ejemplo por fuentes reales.
5. Mantener contenido.html en la misma carpeta que index.html.
6. Publicar todos los archivos en GitHub.

Punto adicional:
La imagen en contenido.html ya tiene la clase image-20, que la centra y le da un ancho de 20%.
